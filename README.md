# UCSC BIT GPA Calculator

Web app for UCSC BIT students to plan GPA outcomes across semesters and years,
check progression/degree rules, and export a report.

## Tech stack

- React + TypeScript
- Vite
- Tailwind CSS

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Notes

- Results shown by the app are for academic planning.
- Official GPA/class results are published by UCSC on their official channels.
